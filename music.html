Coding 練習題目產生器

懶得想要練什麼的時候，就來這裡按一下，隨機幫你丟一題。題目以「敘述」為主，你可以自己選語言實作。

選一個想練習的主題

題目會偏向你目前有在碰的：Java / C / Python / Verilog / Web / 演算法 / PLC。

按「丟一題」來產生一個練習題目。
建議：開好 VS Code / ISE / Vivado / IntelliJ，一次專心解一題，不一定要寫到完美， 但至少把 input / output 跑起來。

練習方向建議

  • 演算法：array 處理、字串、stack / queue、DFS / BFS
  • Java：REST API、小型 service、例外處理、單元測試
  • C / 系統：指標、struct、檔案 IO、簡單資料結構
  • FPGA / Verilog：狀態機、debounce、顯示器控制
  • Web：表單處理、登入、查詢頁、簡單前端互動
  • PLC：用線路想怎麼接
如果某題你覺得很好玩，可以把它升級成真正的 Side Project， 例如變成一個工具網站或是小遊戲。
